#  Strong Password Generator Web App

A **Cybersecurity Mini Project** built using **Python Flask** that generates secure passwords with customizable options.
The application helps users create **strong, random passwords** to improve online security.

---

##  Features


*  **Custom Password Length**
*  **Option to Include Numbers**
*  **Option to Include Symbols**
*  **Show / Hide Password**
*  **Copy Password to Clipboard**
*  **Password Strength Meter with Colored Bar**
*  **Separate Password History Page**
*  **Fast & Lightweight Web Application**

##  Installation

### 1️ Clone the Repository

```bash
git clone https://github.com/yourusername/password-generator.git
cd password-generator
```

### 2 Install Dependencies

```bash
pip install flask
```

### 3️ Run the Application

```bash
python app.py
```

### 4️ Open in Browser

```
http://127.0.0.1:5000
```

---

## 💻Usage

1. Enter the **password length**.
2. Select options to **include numbers or symbols**.
3. Click **Generate Password**.
4. View the generated password.
5. Use **Show / Hide** to reveal or mask the password.
6. Click **Copy Password** to copy it to clipboard.
7. View all previously generated passwords in the **History Page**.
